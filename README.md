# Android Mini Apps

A collection of beginner to intermediate Android applications built with Java/Kotlin, covering core Android development concepts.

## Apps

| App | Description |
|-----|-------------|
| **Calci** | A clean calculator app with basic arithmetic operations |
| **DOBCal** | Date of Birth calculator — computes exact age and upcoming birthday countdown |
| **First** | Hello World and layout exploration — first Android project |
| **FirstApp** | Extended first app with intent navigation and UI components |

## Getting Started

### Prerequisites
- Android Studio (Electric Eel or newer)
- Android SDK 21+

### Build

1. Open Android Studio
2. Open any app folder as a project (e.g. `Calci/`)
3. Let Gradle sync complete
4. Connect a device or start an emulator → click **Run**
